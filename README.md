 # AA-HUD — Acacia Avenue Heads-Up Display

   Persistent desktop information banner for Windows 10/11 workstations and Window Server 2019/2022/2025.

   AA-HUD displays a continuous, always-on banner at the top of every monitor
   showing user identity, system information, and organizational context —
   designed for enterprise environments that require persistent visual awareness
   at the desktop level.

   ---

   ## Download

   👉 [Latest Release — v1.0.067.4](https://github.com/acaciaavellc/Heads-Up-Display/releases/latest)

   Includes signed MSI installer and Group Policy templates (ADMX/ADML).

   ---

   ## Features

   - Native C++ — no .NET runtime required
   - Multi-monitor support
   - Active Directory token substitution (`{DISPLAY_NAME}`, `{TITLE}`, `{COMPUTER_NAME}`, and more)
   - Group Policy managed (ADMX/ADML templates included)
   - Authenticode signed
   - Windows Task Scheduler integration (launches at every user logon)

   ---

   ## Requirements

   - Windows 10 (22H2+), Windows 11, or Windows Server 2019/2022/2025 — x64
   - Active Directory domain-joined workstation or server
